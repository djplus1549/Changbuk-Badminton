Firebase 연결 설정

처음 한 번만 입력하면 기기에 저장됩니다

STEP 1 — Firebase 프로젝트 만들기

Realtime Database 생성

1. console.firebase.google.com 접속
2. 프로젝트 추가 → 이름 입력 → Google Analytics 끄기 → 만들기
3. 왼쪽 메뉴 빌드 → Realtime Database → 데이터베이스 만들기
4. 위치: asia-southeast1 → 테스트 모드 → 사용 설정

STEP 2 — 웹 앱 등록 & config 복사

firebaseConfig 값 복사

1. 프로젝트 홈 → </> (웹 앱) 클릭 → 앱 닉네임 입력 → 등록
2. 표시되는 firebaseConfig 에서 아래 두 값을 복사해서 입력하세요

DATABASE URL
API KEY
창북중학교 여중부
